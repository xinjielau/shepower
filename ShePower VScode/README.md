# 三立女人 Brand Style Website

This version is designed to feel more like a brand website:
- prettier editorial layout
- soft animation
- gradient sections
- magazine-style photo layout
- more photos
- video sections
- no Register button
- no Join Us button

## Run
npm install
npm run dev

## Compatible Node
This version uses Next 13.5.6 to work more safely with Node 18.15.

## Replace later
- `data/siteData.js` for photos, videos, members, activities
- `components/Layout.js` for social links
